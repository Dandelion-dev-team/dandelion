Description
 
The timescale for this project identifies three main phases of which the latter two are based on agile iterative methods. The initial design phase estalishes acceptance criteria for the whole projet. It also defines the main API on which the rest of the system depends.

During the development phase, the major elements of the system will be progressed in parallel as far as possible. Towards the end of the development phase, features such as the schools dashboard and data visualisation methods will require the input of the whole team. The development phase also includes thorough integration testing.

The user testing phase requires the input of early-adopter schools who are willing to put the system through its paces before the wider release on 18 Apr 2022.
